# Trabalho Extensionista — Faculdade

Projeto extensionista desenvolvido com foco em incentivo à atividade física e combate ao sedentarismo, oferecendo uma proposta de treino personalizado por meio de uma interface web simples e acessível.

## 📖 Sobre o projeto

O **Trabalho Extensionista — Faculdade** é uma aplicação web desenvolvida com a proposta de incentivar pessoas sedentárias a iniciarem uma rotina de exercícios por meio de uma experiência simples, direta e acessível.

A ideia central do projeto é oferecer um fluxo inicial de entrada e cadastro para o usuário, servindo como base para uma plataforma voltada à personalização de treinos e promoção de hábitos mais saudáveis.

## ✨ Funcionalidades

- Tela inicial de boas-vindas
- Fluxo para usuários iniciantes ou já cadastrados
- Cadastro de usuário
- Entrada/login no sistema
- Interface voltada para incentivo à atividade física
- Estrutura de navegação entre telas
- Organização do projeto em HTML, CSS e JavaScript
